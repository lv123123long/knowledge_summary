# 换源

```
docker pull dockerpull.org/jenkinszh/jenkins-zh
```

在docker pull后先拼接[镜像源](https://so.csdn.net/so/search?q=%E9%95%9C%E5%83%8F%E6%BA%90&spm=1001.2101.3001.7020)域名，后面拼接拉取的镜像名
* dockerpull.org 是域名
* jenkinszh/jenkins-zh   镜像名*
